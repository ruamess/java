## Цель работы:

Ознакомиться с основами языка программирования Java, изучить структуру простой Java-программы и приобрести первичные навыки работы в интегрированной среде разработки Eclipse.

## Задачи лабораторной работы:

установить и настроить среду разработки Eclipse;
создать Java-проект и класс;
изучить структуру Java-программы;
реализовать и запустить простую Java-программу;
закрепить понимание метода `main()`.

## Вопросы
1. Java - объектно-ориентированный язык программирования со статической типизацией созданный, 23 мая 1995 года компанией Sun Microsystems. Слоган и основная суть - это "Пиши код один раз, запускай где угодно".
2. main() - входная точка в программу.
3. Класс - это шаблон (код) который описывает методы, данные будущих объектов. Объект это экземпляр класса.
4. Среда разработки Eclipse - как и другая среда разработки (IDE) используется для написания, отладки, дебага кода. Но основное отличие ее от других это модульность.
